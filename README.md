## 🚀 项目介绍

Element Plus Lego 是一个基于 Vue3 + Element Plus 的开箱即用的二次封装组件库

## 🌐 演示地址

[查看文档](https://element-plus-lego.vercel.app/) ｜ [完整示例](https://bricklayer-admin.vercel.app)

## ✨ 特性

- 🎯 **开箱即用** - 提供完整的项目结构和配置
- 🧩 **组件封装** - 深度封装常用业务组件，减少重复代码
- 📱 **响应式设计** - 支持多种设备屏幕适配
- 🔧 **Hook 封装** - 提供常用业务逻辑 Hook
- 📦 **示例完善** - 每个组件都有对应的使用示例
